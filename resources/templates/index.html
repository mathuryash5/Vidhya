Vidhya

Vidhya is a solution to generate insights from the large amount of unstructured data. It uses deep learning and natural language processing techniques to extract information from the text. It aims to help researchers and medical professionals get useful information from vast unstructured data.

Features of Vidhya

VidhyaDiscover

A Knowledge Graph to visualize relations between entities present in the Microsoft CORD-19 dataset. The graph contains seven different types of entities. Offers the ability to predict new links between entities thereby helping researchers find new relations between various entities.

Vidhya Paper Recommender

Identifies and suggests similar relevant biomedical articles on the topic of interest. Provides top “n” relevant biomedical articles to the users. Provides a way to customize the weights of semantic search based on title of a paper or abstract of a paper.

AskVidhya

AskVidhya enables both the researchers and common man to query the corpus using natural language questions and retrieve answers. AskVidhya provides provide top “n” answers along with the source biomedical article to the users
